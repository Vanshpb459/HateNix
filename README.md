# HATEnix 🚫💬  
An AI-powered hate speech detection and moderation system for online comments.
## 📌 About the Project

HATEnix is an intelligent hate speech detection system designed to identify and moderate hateful, abusive, and offensive comments on online platforms such as YouTube.  

The system uses Natural Language Processing (NLP) and Machine Learning models to analyze user comments and automatically classify them as hateful or non-hateful, helping creators maintain a safe and positive online environment.
## ✨ Features

- 🔍 Detects hate, abusive, and offensive comments
- 🤖 Uses NLP and Machine Learning models
- 📊 Classifies comments in real-time
- 🛡️ Helps automate comment moderation
- 🔑 Secure API-based comment fetching (YouTube API)
- 🧠 Supports custom-trained models

## 🛠️ Tech Stack

- **Programming Language:** Python  
- **Libraries:** Pandas, NumPy, Scikit-learn  
- **NLP Models:** RoBERTa  
- **Machine Learning:** Random Forest Classifier  
- **API:** YouTube Data API  
- **Tools:** Git, GitHub

## ⚙️ How It Works

1. Fetches comments using the YouTube Data API  
2. Preprocesses text using NLP techniques  
3. Converts text into numerical features  
4. Classifies comments using trained ML models  
5. Flags or removes hateful comments automatically
